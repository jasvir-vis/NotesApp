# 📚 Notes Management System

A full-stack Notes Management System developed for educational institutions, enabling faculty members to upload, organize, and manage study materials efficiently. The platform supports topic-wise notes management, multiple PDF uploads, and secure user authentication.

## 🚀 Features

### 👨‍🏫 Faculty Module

* Secure login and authentication
* Create and manage topics
* Upload multiple PDF notes
* Update and delete study materials

### 📂 Notes Management

* Topic-wise organization of notes
* Multiple PDF upload support
* Easy access to study resources
* Efficient content management

### 🔐 Authentication & Security

* JWT-based authentication
* Protected routes and APIs
* Role-based access control

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/jasvir-vis/notes-management-system.git
cd notes-management-system
```

### Install Dependencies

```bash
npm install
cd client
npm install
```

### Run Backend

```bash
npm run dev
```

### Run Frontend

```bash
cd client
npm run dev
```
